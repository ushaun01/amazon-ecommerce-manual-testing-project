🛒 Amazon Manual Testing Project

🌍 Project Overview

This repository contains a Manual Testing Project for Amazon (https://www.amazon.com)

It demonstrates the end-to-end e-commerce testing flow covering everything from user login to product ordering.

The project includes detailed test documentation — Test Plan, Test Scenarios, Test Cases, Execution Report, Defect Report, and RTM.

🎯 Objective

To validate the Amazon shopping workflow through manual testing, ensuring smooth functionality, usability, and reliability.

🔍 Key Testing Modules

👤 User Login

🔎 Product Search

🛍️ Add to Cart Functionality

📦 Order Placement
